======================
Supported Data Formats
======================

Cross-sections
~~~~~~~~~~~~~~

Supported formats are::

- ``.pickle`` *Taurex2* pickle format
- ``.hdf5``, ``.h5`` Taurex3

Lightcurve Data
~~~~~~~~~~~~~~~

``pickle`` files.

Collisionally Induced Absorption
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Either ``pickle`` files or HITRAN ``cia`` files are supported
