.. _planet:

============
``[Planet]``
============

This header is used to define planetary properties. Currently, only ``planet_type = simple``
is supported and must be included.

The variables that can be defined are:
    ``mass``
        - float
        - Mass in Jupiter mass
        - Default: ``mass = 1.0``
    
    ``radius``
        - float
        - Radius in Jupiter radius
        - Default: ``radius = 1.0``
